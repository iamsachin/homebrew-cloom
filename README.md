# Homebrew Tap for Cloom

Open-source screen recorder for macOS.

## Install

```bash
brew tap iamsachin/cloom
brew install --cask cloom
```

## Update

```bash
brew upgrade --cask cloom
```

## Requirements

- macOS 26.0 (Tahoe) or later
- Apple Silicon (arm64)

## About

Cloom is a free, open-source Loom alternative for macOS. See the [main repository](https://github.com/iamsachin/cloom) for more information.
